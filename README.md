# 电商后台管理项目前后端
``` js
 前端基于 vue2 + axios + element UI 构建
 后端基于 node + mysql 构建
```


## 启动项目

进入项目打开终端执行以下命令安装依赖包

### `npm start / yarn`

运行项目
### `npm run start / yarn start`

### 运行成功后打开 locaohost: 3000 即可
